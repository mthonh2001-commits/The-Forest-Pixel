# 🌐 Web Version - The Forest-Pixel

## Chạy Web Version

### Local Development:
```bash
cd web/
python -m http.server 8000
# Sau đó mở: http://localhost:8000
```

### Trên Chromebook/Điện thoại:
1. Mở trình duyệt Chrome
2. Vào: https://mthonh2001-commits.github.io/The-Forest-Pixel
3. Chơi game! 🎮

---

## 🕹️ Điều khiển:
- **W A S D** - Di chuyển
- **SPACE** - Tương tác Campfire
- **R** - Reset Stats

---

## 📋 Yêu cầu:
- Trình duyệt hiện đại (Chrome, Firefox, Safari, Edge)
- Kết nối Internet (lần đầu tiên)
- Có thể chơi Offline sau khi load

---

## ⚙️ Cách Build:

### Trên Windows/Mac/Linux:
1. Mở Godot
2. Project → Export
3. Chọn HTML5
4. Export vào thư mục `web/`
